What is Tourney?

Tourney is a lightweight piece of software for running chess tournaments between engines. Tournaments can be watched over the web by anybody. With Tourney, multiple computers can participate in the same tournament.

Who is Tourney for?

Those who develop chess engines and those who maintain chess engine ranking lists may be interested in how Tourney can help them. However, any chess engines enthusiast could find Tourney useful.

Project Started

7/14/2014

Over the course of working on my chess engine, I primarily used Arena 3.0 and ChessGUI to test and tune. While they provided a lot of features that I enjoyed, I constantly found myself wanting more. Particularly:

  • Less overhead for running a chess game. One second games should take one second.
  • More than one computer should be able to participate in the tournament. Some tournaments can take several weeks or months on a single computer (as is the case with some tuning techniques). What about friends and family who have computers idling all day while they are at work? Think Prime95 but for chess tournaments!
  • Anybody should be able to watch the tournament from anywhere. By using a web interface to broadcasting games, progress of the tournament can be checked on any OS (including phones and tablets) from any location without special software.
  • Data should be easy to view. For long tournaments, finding engine output for a particular move of a particular game can be time consuming, even daunting. Instead, all of the tournament data (results, engine logs, trends, etc.) should be cross referenced.
